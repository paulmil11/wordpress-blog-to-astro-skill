# WordPress to Astro Migration Skill

A Claude Code skill that provides battle-tested scripts, patterns, and troubleshooting guidance for migrating WordPress blogs to Astro — distilled from real-world migrations.

## Install

```
/install github:paulmil11/wordpress-blog-to-astro-skill
```

## What's Included

| File | Description |
|------|-------------|
| **`skills/wordpress-to-astro/SKILL.md`** | Main guide — end-to-end migration workflow, content modeling, and best practices |
| **`skills/wordpress-to-astro/migration-scripts.md`** | Ready-to-use script templates for exporting, transforming, and importing content |
| **`skills/wordpress-to-astro/common-issues.md`** | Troubleshooting reference for the most frequent migration pitfalls and their fixes |
